# Tkinter Calculator (Python GUI)

A simple **GUI Calculator** built using **Python and Tkinter**.  
It performs basic arithmetic operations like **addition, subtraction, multiplication, and division** with a clean dark-themed interface.

---

## Features

- Graphical User Interface using **Tkinter**
- +, -, *, / Supports basic arithmetic operations
- Uses Python `eval()` for expression evaluation
- Handles invalid expressions gracefully
- Dark theme UI
- Dynamic button generation using loops

---

## Project Structure
- calc.py # Main Python file
- README.md # Project documentation

## How to Run the Project
### Prerequisites
- Python **3.x** installed
- Tkinter (comes pre-installed with Python)

- Check Python version: `python --version`
- Run the Calculator: `python calc.py`

- The calculator window will open immediately...
